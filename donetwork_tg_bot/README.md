# DONetwork Telegram Bot

这是一个为 DONetwork 社区设计的 Telegram 智能回复机器人，具备以下功能：

- **智能问答**：基于提供的知识库回答用户问题，并保持专业、风趣的风格。
- **自动语言切换**：根据用户消息语言（中/英）自动切换回复语言。
- **置信度判断**：低置信度回复将通知管理员。
- **每日数据统计报告**：每天定时生成社区活跃度、人数变化、讨论话题和低置信度问题报告，并私聊发送给管理员。
- **实时内容审核**：自动检测 FUD、垃圾广告和人身攻击，并根据违规次数进行阶梯式惩罚（删除消息、禁言）。

## 部署说明

### 环境变量
部署前，请确保设置以下环境变量：

- `TELEGRAM_BOT_TOKEN`: Telegram 主机器人的 Bot Token。
- `ADMIN_TELEGRAM_ID`: 管理员的 Telegram 用户 ID，用于接收低置信度问题和每日报告。
- `REPORT_BOT_TOKEN`: 用于发送每日报告和违规通知的另一个 Telegram Bot Token（建议使用单独的 Bot 以避免速率限制）。
- `OPENAI_API_KEY`: OpenAI API Key，用于 AI 问答、话题总结和内容审核。

### 权限要求
- **主机器人**：需要被拉入群组并设置为管理员，拥有“删除消息”和“封禁用户”的权限。
- **报告机器人**：只需要能够私聊管理员即可。

### 启动命令
`worker: python donetwork_bot.py`
